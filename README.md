Тема:  
Анализ факторов, влияющих на цену и длительность песен в Apple Music.

Гипотеза 1. Песни с пометкой Explicit стоят дороже «чистых» версий.  
*Этапы:*  
- Посчитать среднюю цену для explicit, clean, notExplicit.  
- Построить столбчатую диаграмму средних цен и круговые диаграммы долей цен (0.69, 1.29) в каждой группе.  
- Сравнить доли низких цен: если у clean доля $0.69 заметно выше – гипотеза подтверждается.

Гипотеза 2. Треки в жанрах Hard Rock и Metal длиннее поп- и хип-хоп-песен.  
*Этапы:*  
- Оставить треки жанров Hard Rock, Metal, Pop, Hip-Hop/Rap.  
- Перевести длительность в минуты, найти среднюю для каждого жанра.  
- Построить столбчатую диаграмму средних длительностей.  
- При желании добавить разбивку по годам и линейный график, чтобы проверить устойчивость разницы во времени.

Гипотеза 3. В 2018–2023 гг. медианная цена трека снизилась по сравнению с 2000–2010 гг. из-за роста доли синглов и делюкс-изданий.  
*Этапы:*  
- Сравнить медианную цену в периоды 2000–2010 и 2018–2023 (столбчатая диаграмма).  
- Построить график среднегодовой цены за весь период.  
- По названию коллекции найти синглы (Single) и делюкс-версии (Deluxe), посчитать их долю по годам.  
- Сопоставить график доли таких релизов с графиком цены: если при росте доли синглов/делюксов цена падает – связь вероятна.
